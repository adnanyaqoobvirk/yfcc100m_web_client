API

The following API's are available for this service:

1. http://45.63.6.234:3000/api/categories

2. http://45.63.6.234:3000/api/images/[category]/[limit]

With the first url you can obtain the all the image categories and with the second url you can obtain the images based on the above categories and with specifying the number of images in the limit parameter.
